Sublimea安装使用

发表于 2019-02-17 分类于 随笔

sublime介绍

sublime是一个代码编辑器，可以编写HTML，php，js，css等文件

sublime 有哪些优点：

  • 跨平台
  • 扩展性强
  • 提交小，运行快
  • 支持编辑功能，可以在控制台查看输出
  • 支持大量的插件

sublime 安装：

下载 ：http://www.sublimetextcn.com/3/ 对应系统的版本

安装：必须安装在电脑硬盘中，路径不能触限中文字符

使用sublime创建一个HTML文件：

Ctrl+N 新建文件，输入index.html

ctrl+S 保存文件，生成index.html文件

新建项目：

菜单栏单击项目-》下拉框中选择我【文件夹添加到项目】-》选择文件夹

sublime常用命令

  • Ctrl + Shift + D—复制当前行到下一行
  • Ctrl + N —新建
  • Ctrl+Shift+K—快速删除整行
  • Ctrl+鼠标左键单击—集体输入
  • Ctrl+H—查找替换
  • Ctrl+/–注释单行
  • Ctrl+Shift+/—注释多行
  • F11—全屏
  • Ctrl + 滚动　–字体变大/缩小

sublime 插件安装

  • 安装Package control（插件安装包管理器）

    https://blog.csdn.net/kongguyoulan523/article/details/51144254

  • 汉化插件 ChineseLocalization

    ctrl+shift+p

    在命令框中选中 package control：install package 选中它按回车

    在命令框中输入 ChineseLocalization，选中插件，按回车

  • ConvertToUTF8自动把文件编码方式转为UTF-8

    ctrl+shift+p

    在命令框中选中 package control：install package 选中它按回车

    在命令框中输入 ConvertToUTF8，选中插件，按回车

  • Emmet快速编写HTML、CSS标记的插件

    常用快捷键

    • !+tab 快速生成html5骨架

    • 其他常用快捷键

  • IMESupport 是用来支持输框跟随光标显示的插件
  • SublimeCodeIntel代码提示插件
  • SidebarEnhancements 侧边栏插件，让侧边栏功能更丰富
  • markdown

删除插件

ctrl+shift+p

在命令框中选中 package control：remove package选中它 按回车

在命令框中输入插件名称，选中插件，按回车
